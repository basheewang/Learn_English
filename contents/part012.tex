\section{状语从句}

\subsection{when引导时间状语从句}

用when引导时间状语从句时，需要注意以下几个方面：
\begin{enumerate}
  \item 从句的谓语动词既可以是延续性动词，也可以是短暂性动词。从句的动作可以和主句的动作同时发生，也可以先于主句的动作发生，此时相当于after。
  \item 当when从句的谓语动词是延续性动词时，可与while互换。比如:When/While the students were listening attentively, the teacher explained the difficult points.当学生们在认真听讲时，老师讲解了难点内容。
  \item 当when引导的时间状语从句置于主句之后，且从句的谓语动词又是短暂性动词时when 表示“那时，这时”，有事发突然的意味。比如:I was writing an essay when my computer suddenly broke down. 我正在写论文，这时电脑突然出故障了。
\end{enumerate}

\subsection{as引导时间状语从句}

as常用来引导时间状语从句，其用法汇总如下：
\begin{enumerate}
  \item 表示“当……时”，此时从句的谓语动词要用延续性动词，一般不使用be动词或表示心理活动及状态的词。比如:As I was reading a thought-provoking article, I came up with some new ideas for my essay. 当我在读一篇发人深省的文章时，我为我的论文想的了一些新点子。
  \item as还可表示“一边……一边……，随着”，此时从句的谓语动词没有要求。比如:As the population grows, the demand for housing increases significantly. 随着人口增长,住房需求显著增加。
\end{enumerate}

\subsection{until/till引导时间状语从句}

until 和 till 都表示“直到……”，当置于句首时，多用 until。
\begin{enumerate}
  \item 用在肯定句中时，主句的谓语动词多为延续性动词，表示“直到……为止”，表示主句动作一直延续到until/till从句表示的时间才终止。比如:She studied until/till the library closed. 她一直学习到图书馆关门。
  \item 用在否定句中时，主句的谓语动词通常是终止性动词，表示“直到……才”。比如:They didn't start the project until/till they received the necessary funds. 他们直到收到必2.要的资金才开始这个项目。
\end{enumerate}

\subsection{before引导时间状语从句}

before用作连词常用来引导时间状语从句，其用法如下:
\begin{enumerate}
  \item 表示“在……之前”或“还没来得及……就……”，强调从句动作发生之前，主句动作就已经发生了。比如:The plane took off before I reached the airport. 我还没到机场，飞机就起飞了。
  \item 用于句型“It be+一段时间+before+从句”中，表示“过了一段时间之后才…”强调beore之后的动作来之不易。比如: It will be weeks before the repairs are completed. 要过几周维修才能完成。
\end{enumerate}

\subsection{地点状语从句}

\begin{enumerate}
  \item 地点状语从句的常用引导词有where (在……地方)、wherever (无论在什么地方)、anywhere (无论何处)以及everywhere (到处)等。比如:You can set up your tent
        wherever there is a flat ground. 你可以在任何有平地的地方搭帐篷。
  \item where既可引导定语从句，又可引导地点状语从句，判断它引导的是哪类从句的关键是看 where 前面是否有被修饰的、表示地点的先行词，比如:
        \begin{itemize}
          \item Put the books where they were. 把书放回原位。 (地点状语从句)
          \item Put the books on the shelf where you usually keep them. 把书放在你通常放书的那个书架上。 (定语从句)
        \end{itemize}
\end{enumerate}

\subsection{because引导原因状语从句}

because 意为“因为”，与其他原因状语从句引导词相比，语气最强，使用because时需注意以下几个方面：
\begin{enumerate}
  \item because 可用来回答 why 提出的问题，since 和 as 都不可以。
  \item 短语 because of也表示“因为……”，但是不能直接引导从句或接that引导的从句，它后面接名词或名词性短语，比如:The flight was canceled because the fog was heavy. = The flight was canceled because of the heavy fog. 航班因为大雾取消了。
  \item 汉语中常说“因为……所以……”，但英语中 because和so不能连用。
  \item because 还可用于引导表语从句，比如:That is because I'm too busy. 那是因为我太忙了。
\end{enumerate}

\subsection{as引导原因状语从句}

\begin{enumerate}
  \item as可用来引导原因状语从句，语气比because和since 都弱，较为口语化，一般表示原因比较明显或是双方都知道的事实。从句通常放在句首，有时也可放在句末，比如:As the deadline is approaching, we need to speed up our work. 由于截止日期临近，我们得加快工作进度。
  \item as引导原因状语从句时，有时需要倒装，比如:Excited as he was, he hardly fell asleep. 由于太激动了，他无法人眠。
\end{enumerate}

\subsection{given that引导原因状语从句}

\begin{enumerate}
  \item given that在引导原因状语从句时，意为“鉴于，考虑到”。比如:Given that the new technology is not fully tested, we need to be cautious about its application. 考虑到这项新技术尚未完全测试，我们对其应用需持谨慎态度。
  \item given在单独使用时，还有“假设，如果”的意思，后面常跟名词或名词词组。比如:Given the opportunity, I would choose to study abroad. 假如有机会，我会选择出国留学。
\end{enumerate}

\subsection{其它原因状语从句常见引导词：now that/in that/seeing that\(\ldots\)}

在引导原因状语从句时，除了前文提到过的because、since、as、given that等，还有一些常见的词组作为引导词，如：now that (既然), seeing that (由于), in that (因为), considering that (考虑到), for the reason that (因为), on the grounds that (因为)等，如：
\begin{itemize}
  \item This design is unique in that it combines traditional and modern elements. 这个设计很独特，因为它融合了传统与现代元素。
  \item Considering that he has no experience, he has done a pretty good job. 考虑到他没有经验，他已经做得相当不错了。
\end{itemize}

\subsection{目的主语从句}

\begin{enumerate}
  \item so that 在引导目的状语从句时，意为“为了，以便”，句中常有情态动词出现，that有时会省略。比如:She set an alarm early so (that) she wouldn't be late for the important meeting. 她早早地设了闹钟，以免在重要会议上迟到。
  \item 在引导目的状语从句时，除了so that，还有一些常见的引导词，如in case、in order that、for fear that 等。比如:The government has implemented various policies in order that the economy can recover from the recession. 政府已经实施了各种政策，以便经济能够从衰退中复苏。
\end{enumerate}

\subsection{in case引导目的状语从句}

in case 可用作短语连词，引导状语从句：
\begin{enumerate}
  \item 当表示“以免，免得，以防”时，引导目的状语从句。比如:Wear a mask in public places in case you catch a virus. 在公共场所戴口罩，以防感染病毒。
  \item 当表示“如果，万一”时，引导条件状语从句。比如:In case it rains tomorrow, we can hold the event indoors. 万一明天下雨，我们可以在室内举办活动。
\end{enumerate}

\subsection{条件状语从句中“主（句）将（将来时、祈使句）从（句）现（现在时）”}

在含有条件状语从句的复合句中，如果主句是一般将来时，或者主句是祈使句，或使用了情态动词，从句要用一般现在时，即“主将从现”，比如:
\begin{itemize}
  \item Unless you hurry up, you won't catch the last bus. 除非你快点，否则就赶不上末班车了
  \item Provided that you grasp the key concepts of this theory, apply them to real-life situations for practice. 假如你掌握了这个理论的关键概念，将它们应用到实际生活中进行实践。
  \item If it is sunny this weekend, we can go on a picnic. 如果这个周末天气晴朗，我们可以去野餐。
\end{itemize}

\subsection{unless引导条件状语从句}

unless意为“除非”，常用来引导条件状语从句，在条件状语从句中使用unless时需注意以下几个方面：
\begin{enumerate}
  \item 可以和“if \(\ldots\) not”互换，但是与“if \(\ldots\) not”相比，unless 的语气更强烈一些。比如:You will not know the answer unless you listen carefully. = You will not know the answer if you don't listen carefully. 除非你认真听讲，否则你不知道答案。
  \item 在unless引导的从句中，谓语动词不能再用否定形式。比如:
        \begin{itemize}
          \item Unless you don't study hard, you will fail. (错误用法)
          \item Unless you study hard, you will fail. (正确用法)
        \end{itemize}
  \item if引导的条件状语从句可以用虚拟语气，但是unless引导的条件状语从句通常不使用虚拟语气。
\end{enumerate}

\subsection{条件状语从句引导词：if/unless/as (so) long as/once/in case/supposing/provided\(\ldots\)}

常用于引导条件状语从句的词或词组有:if (如果), unless (除非), as/so long as (只要), once (一日), in case (万一), on condition that (条件是), supposing (that) (假如), provided (that) (假如), only if (只要)等。

\subsection{only if引导条件状语从句}

\begin{enumerate}
  \item only if表示“只要，只有当”，在使用时需注意，如果only if引导的条件状语从句位于主句之前，主句需要用部分倒装 (情态动词/助动词/be动词 + 主语 + 动词原形);而如果该从句位于主语之后，主句使用正常语序。比如:Only if you fully understand the cultural background can you accurately interpret the meaning of this idiom. 只有当你充分了解文化背景，你才能准确解读这个习语的含义。
  \item 还有一个词组和 only if很相似，即if only，要注意区分。if only 意为“要是……就好了”，用于虚拟语气，表达一个不可能实现的愿望或假设。比如:If only I had spent more time improving my writing before the exam. 我要是在考试前多花些时间提升写作就好了。
\end{enumerate}

\subsection{if条件状语从句中的省略}

\begin{enumerate}
  \item 在if条件状语从句中，当从句的主语和主句的主语一致 (或从句主语为it)，且从句谓语中含有 be 动词时，从句的主语和 be 动词可以一同省略。比如:If (it is) heated, water will turn into steam. 如果受热，水会变成蒸汽。
  \item 常见的if条件句省略形式有: if necessary (如果有必要的话), if true (假如是真的), if not (如果不这样的话), if any(如果有一些的话), if possible (如果有可能的话), if so (假如是这样的话), if ever (如果曾经有过的话), if anything (要说真有什么的话)等。
\end{enumerate}

\subsection{although/though引导让步状语从句}

\begin{enumerate}
  \item although和though都能引导让步状语从句，表示“虽然”，可以与yet、still或nevertheless等词连用，但二者都不能与but连用。比如:Although/Though the cost of living in this city is high, many people still choose to settle here. 尽管这座城市的生活成本很高，许多人还是选择在这里定居。
  \item 二者通常可以互换使用，但though引导的让步状语从句可以以部分倒装的形式出现，although 却不能这样使用。比如:Busy though she is, she always finds time to volunteer at the local community center. 尽管她很忙，但她总是能抽出时间在当地社区中心做志愿者。
\end{enumerate}

\subsection{even though/even if引导让步状语从句}

\begin{enumerate}
  \item even tough和even if都表示“即使…”，两者经常互换使用。比如:We will continue with the experiment even though/even if we face numerous difficulties. 即使面临诸多困难，我们仍将继续进行实验。
  \item even though和though都可以引导让步状语从句，但是even though可引导事实性让步 (已知事实) 或假设性让步 (极端/非典型情况) ，带有强调的语气，而though通常引导事实性让步。比如:
        \begin{itemize}
          \item Even though the new policy appears beneficial, it may face resistance from some groups. 即使新政策看似有益，但它可能会面临一些群体的抵制。
          \item Though he has a solid foundation in English, he still needs to practice more speaking尽管他英语基础扎实，但仍需多练习口语。
        \end{itemize}
\end{enumerate}

\subsection{“疑问词（when/where/what/which/who）\(-ever\)”引导让步状语从句}

使用“no matter + 疑问词”或“疑问词-ever”引导让步状语从句时，通常强调主句的内容为事实或不可改变的结果：
\begin{enumerate}
  \item “no matter + 疑问词”引导让步状语从句时，可转化成“疑问词-ever”，常用于此结构的疑问词有what、which、who、where、when、how等。比如:No matter where (=Wherever) you study, self-discipline is crucial.无论你在哪里学习，自律都至关重要
  \item whenever和wherever还可分别引导时间和地点状语从句。
  \item 当“疑问词-ever”引导名词性从句时，不能与“no matter +疑问词”互换。
\end{enumerate}

\subsection{常用让步状语从句引导词}

常用于引导让步状语从句的单词或词组有:
\begin{enumerate}
  \item 单词：as (虽然), although/though (虽然), while (虽然)
  \item 词组：even if/even though (即使), whether \(\ldots\) or \(\ldots\) (不管……还是……), whether \(\ldots\) (or not) (不管是否),
        no matter (who/what/where/when/how) = (who/what/where/when/how) + ever无论 (谁/什么/哪里/何时/怎样)等。
\end{enumerate}
